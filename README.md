# Site Kantrol

Site institucional/portfólio em HTML, CSS e JavaScript puro.

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Para publicar, envie toda a pasta para sua hospedagem ou GitHub Pages.
3. Edite os contatos no final do arquivo `index.html`:
   - Link do WhatsApp: `https://wa.me/5511999999999`
   - E-mail: `contato@kantrol.com.br`

## Estrutura

- `index.html`: estrutura da página.
- `css/style.css`: visual, responsividade e paleta de cores.
- `js/script.js`: menu mobile, animações e link ativo.
- `assets/`: imagens com fundo removido e logo.
